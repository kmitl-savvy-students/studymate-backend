# StudyMate Backend

StudyMate is a backend service built with ASP.NET Core, designed to manage and facilitate study groups and educational
content. This project uses PostgreSQL for database management and includes support for Swagger for API documentation.

## Getting Started

### Prerequisites

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [PostgreSQL](https://www.postgresql.org/download/)

### Installation

Install the required .NET packages:

```sh
dotnet restore
```

### Running the Application

To run the application run using Docker:

The application will start on `http://localhost:5000`.

### API Documentation

Swagger UI will be available at `https://localhost:5000/swagger` for API exploration and testing.
